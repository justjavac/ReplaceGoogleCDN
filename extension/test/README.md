4. [`cdnjs.cloudflare.com` 替换为 `cdnjs.loli.net` 打开  `https://cdnjs.cloudflare.com/` 查看结果  ](https://cdnjs.cloudflare.com/ajax/libs/reveal.js/4.1.2/reveal.min.css)
